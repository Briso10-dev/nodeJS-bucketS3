
# Uploading files

Designing a file upload process into a bucket S3 through AWS SDK for javascript. 


## Tech Stack

**Server:** NodeJS


## Installation

Install my-project with yarn

```bash
  yarn install
```
Install the neccessary dependency

```bash
  yarn add @aws-sdk/client-s3
```
